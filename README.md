# McValley Client Pack

ตัวเกมและไฟล์มอดล่าสุดสำหรับเซิร์ฟเวอร์ McValley

## วิธีเข้าเล่น

1. ดาวน์โหลด `McValley_Setup.exe` จากหน้า Releases แล้วติดตั้งครั้งเดียว
2. กดไอคอน **McValley** บน Desktop

ตัวเปิดเกมจะโหลดเฉพาะไฟล์ที่เปลี่ยน เปิด Resource Pack ให้อัตโนมัติ และเชื่อมต่อ
`hitza13.thddns.net:5570` เท่านั้น ไม่ต้องดาวน์โหลด ZIP หรือเปิด BAT/PowerShell เอง
