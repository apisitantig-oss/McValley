# McValley Client Pack

ตัวเกมและไฟล์มอดล่าสุดสำหรับเซิร์ฟเวอร์ McValley

## วิธีติดตั้งและเข้าเล่นเกม:
1. ดาวน์โหลดคลังไฟล์นี้แบบ ZIP (ไปที่ปุ่มสีเขียว **Code** -> **Download ZIP**)
2. แตกไฟล์ทั้งหมดลงในโฟลเดอร์ตัวเกมหลักของท่าน
3. ดับเบิ้ลคลิกไฟล์ **`Play_McValley_GUI.exe`** เพื่อเข้าเล่นเกมได้ทันที!
   * ตัวลันเชอร์แบบ GUI จะคอยตรวจสอบไฟล์มอดและคอนฟิกใหม่จาก GitHub และทำการดาวน์โหลดอัปเดตให้โดยอัตโนมัติก่อนรันเข้าเกมหลักทันที!
