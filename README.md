# McValley Client Pack

ตัวเกมและไฟล์มอดล่าสุดสำหรับเซิร์ฟเวอร์ McValley

## วิธีติดตั้งและอัปเดตตัวเกม:
1. ดาวน์โหลดคลังไฟล์นี้แบบ ZIP (ไปที่ปุ่มสีเขียว **Code** -> **Download ZIP**)
2. แตกไฟล์ทั้งหมดลงในโฟลเดอร์ตัวเกมหลักของท่าน (ให้ไฟล์ `Play_McValley.bat` อยู่ห้องเดียวกับ `playstardew.exe`)
3. ดับเบิ้ลคลิกไฟล์ **`Play_McValley.bat`** เพื่อเข้าเล่นเกม
   * ตัวสคริปต์จะคอยตรวจสอบไฟล์มอดและคอนฟิกใหม่จาก GitHub นี้ และทำการดาวน์โหลดอัปเดตให้โดยอัตโนมัติก่อนรันเข้าเกมหลักทันที!
